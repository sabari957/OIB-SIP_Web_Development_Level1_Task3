# Temperature Converter

A simple web-based temperature converter that converts between Celsius, Fahrenheit, and Kelvin.

## Features
* Input temperature in Celsius, Fahrenheit, or Kelvin
* Select input unit using radio buttons
* Convert temperature with a single click
* Display converted temperature with correct unit
* Input validation for numbers only

## Usage
1. Enter a temperature value in the input field
2. Select the input unit (Celsius, Fahrenheit, or Kelvin)
3. Click the "Convert" button
4. View the converted temperature

## Code
* `index.html`: HTML structure for the converter
* `style.css`: CSS styles for the converter
